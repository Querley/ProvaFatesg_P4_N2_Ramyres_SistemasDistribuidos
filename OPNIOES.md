# OPNIOES

## Qual tecnologia de comunicação o servidor de cálculo utiliza?

O servidor de cálculo utiliza comunicação via Socket TCP.

---

## Qual protocolo de mensagem o servidor de cálculo utiliza?

O protocolo utilizado é JSON-RPC 2.0.

---

## Qual tecnologia de comunicação o servidor de cálculo utiliza?

A comunicação entre os serviços ocorre utilizando Java RMI e Socket TCP.

---

## Qual estilo arquitetural o projeto web api implementou para expor suas funcionalidades?

O projeto web_api implementa uma arquitetura REST API utilizando Spring Boot.

---

## Opinião sobre o projeto

O projeto apresentou uma boa abordagem sobre sistemas distribuídos utilizando diferentes formas de comunicação entre aplicações, como RMI, Socket e REST. Também foi possível entender melhor conceitos de desacoplamento, comunicação entre serviços e tolerância a falhas.
